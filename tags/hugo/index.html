欢迎来到我的技术博客 这是我使用 Hugo 搭建的个人博客的第一篇文章！ 为什么选择 Hugo？ Hugo 是一个用 Go 语言编写的静态网站生成器，具有以下优点： 速度极快：Hugo 以其生成网站的速度著称 简单易用：配置简单，上手容易 主题丰富：有大量精美的主题可以选择 适合博客：特别适合用来搭建个人博客和文档网站 博客搭建过程 安装 Hugo 创建新的 Hugo 站点 安装 PaperMod 主题 配置博客基本信息 撰写第一篇文章 本地预览和部署 未来计划 在这个博客中，我计划分享以下内容： 技术学习笔记 项目经验分享 实用工具推荐 编程技巧总结 感谢您的访问！如果有任何问题或建议，欢迎留言交流。
